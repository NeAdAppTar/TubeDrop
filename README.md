**TubeDrop** — приложение на C# (WPF) для скачивания видео с YouTube в максимальном качестве. 

---

### Требования
* Установленная [Visual Studio](https://visualstudio.microsoft.com/) (с рабочей нагрузкой ".NET Desktop Development")
* SDK .NET 10.0 или выше

---

### Дальнейшее развитие проекта
* Выбор качества видео
* Выбор папки для сохранения
* Автоматический перехват ссылки
* Скачивание только аудио
